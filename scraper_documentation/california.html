california_scraperR Documentation

Scraper class for general California COVID data

Description

California data scraped from rendered power BI iframe. Testing data is also available at the facility level but will need to be scraped separately. Note that time series data for the state exists for comparison.

Institution Name

The facility name.

Confirmed

The number of confimed cases among residents.

New In Last 14 Days

The number of new cases among residents.

Active in Custody

The number of active cases.

Released While Active

Residents released after testing positive.

Resolved

Recovered cases.

Deaths

Resident Deaths.

Details

The last run of the scraper was on 2021-01-11 and contained the extracted columns: Name, Residents.Confirmed, Residents.Active, Residents.Recovered, Residents.Deaths, State, Date, id, source, jurisdiction. We are missing the following core variables for the analysis: Staff.Confirmed, Staff.Deaths, Staff.Recovered, Staff.Tested, Residents.Tested, Residents.Tadmin, Staff.Negative, Residents.Negative, Staff.Pending, Residents.Pending, Staff.Quarantine, Residents.Quarantine, Residents.Population
