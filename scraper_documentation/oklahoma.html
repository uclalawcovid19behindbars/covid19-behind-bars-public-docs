oklahoma_scraperR Documentation

Scraper class for general Oklahoma COVID data

Description

HTML scraping from a pseudo-table like object. This is really weird to scrape from and the code should be more fail safe.

Details

The last run of the scraper was on 2020-10-19 and contained the extracted columns: Name, Residents.Confirmed, Residents.Recovered, Residents.Quarantine, Staff.Confirmed, Staff.Recovered, Staff.Deaths, Residents.Deaths, State, Date. We are missing the following core variables for the analysis: Staff.Tested, Residents.Tested, Staff.Negative, Residents.Negative, Staff.Pending, Residents.Pending, Staff.Quarantine, Residents.Population
